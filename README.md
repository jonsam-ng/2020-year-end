# react-ts-mobx template

## usage

- install: yarn
- run: yarn start

## techs

- lang: typescript
- framework: react hooks
- style: sacc
- UI: antd
- store: MobX
- router: react-router-dom
- log: sentry
- tools: lodash ahooks react-document-title
- i18n: react-i18next
- animation: ant motion
- test: jest
